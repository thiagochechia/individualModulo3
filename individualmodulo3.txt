3 - COMO ESSAS ENTIDADES ESTAO RELACIONADAS?

TABELA RESILIA:
ID INT é uma PK (Primary Key)
ID COLABORADORES = É composta por duas chaves estrangeiras ( FK )DA TABELA COLABORADORES  : (CPF E NOME)
ID TECNOLOGIA = (FK) da TABELA TECNOLOGIA  (TECNOLOGIA)
ID AREA = (FK) da TABELA AREA  (AREA)
ID PARCEIRA = (FK) composta por duas chaves da TABELA PARCEIRA (ID e NOME)


TABELA COLABORADORES:
ID INT é uma PK (Primary Key)
NOME 
CPF (PK)
ID TECNOLOGIA = (FK) TABELA TECNOLOGIA ( TECNOLOGIA)
ID AREA = (FK) TABELA AREA (AREA)
ID PARCEIRA = (FK) TABELA PARCEIRA (PARCEIRA)


TABELA AREA:
ID INT é uma PK (Primary Key)
AREA
ID TECNOLOGIA = (FK) composta por duas chaves da TABELA TECNOLOGIA (ID + TECNOLOGIA)
ID AREA = (FK) composta por duas chaves da TABELA AREA (ID + PARCEIRA)

TABELA TECNOLOGIA:
ID INT é uma PK (Primary Key)
TECNOLOGIA 
ID AREA = (FK) composta por duas chaves da TABELA AREA (ID + AREA)
ID PARCEIRA = (FK) composta por duas chaves da TABELA PARCEIRA (ID + TECNOLOGIA)

TABELA PARCEIRA:
ID INT é uma PK (Primary Key)
NOME
ID TECNOLOGIA = (FK) Chave estrangeira da TABELA TECNOLOGIA (TECNOLOGIA)
ID AREA = (FK) composta por duas chaves da TABELA AREA (ID+ AREA)
ID COLABORADOR = (FK) composta por duas chaves da TABELA COLABORADOR (CPF+ NOME)
